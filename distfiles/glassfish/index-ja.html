Walbrix 仮想アプライアンス

GlassFish オープンソースエディション

Java EEアプリケーションサーバのリファレンス実装です

オフィシャルサイト: GlassFish Server

GlassFishの管理画面

GlassFishの管理画面は https://{{HOSTNAME}}:4848/です。

管理画面は HTTPと HTTPSを同一のポートで提供する Port Unificationでサービスされており、ブラウザの種類によっては利用できないことがあります。

AJPで Apacheと連携しているので、GlassFishにデプロイした Webアプリケーションには 80番ポートでアクセスできます。

Webコンテンツの編集

Windows共有で下記の共有フォルダへアクセスすることが出来ます。

  • ¥¥{{HOSTNAME}}¥htdocs - 静的コンテンツを配置する場所(このページもここにあります)
    システム上の場所は /var/www/localhost/htdocs です。この場所に配置されていないリソースへのアクセスは GlassFishに転送されます。
  • ¥¥{{HOSTNAME}}¥domain1 - GlassFishのサーバディレクトリです。
    システム上の場所は /usr/local/glassfish4/glassfish/domains/domain1 です。

データベースについて

この仮想アプライアンスで利用出来るデータベースは MySQLと PostgreSQLです。また、GlassFishのクラスパスには H2 Databaseの jarファイルも配置してあります。

MySQLの管理

phpMyAdmin を利用可能です。管理ユーザーIDは root, パスワードは空です（パスワードが空の状態ではプライベートネットワークからのみログイン可能）。MySQLの JDBCドライバは GlassFishのクラスパスに最初から入っています。

PostgreSQLの管理

phpPgAdmin を利用可能です。管理ユーザーIDは postgres, パスワードは空です（パスワードが空の状態ではプライベートネットワークからのみログイン可能）。PostgreSQLの JDBCドライバは GlassFishのクラスパスに最初から入っています。

SSHによるログイン

sshで {{HOSTNAME}} へログインすることもできます。rootの初期パスワードは Walbrixのシリアルナンバー（未登録の場合 "WBFREE01"）です。GlassFishのインストール先は /usr/local/glassfish4 です。
