Apache CouchDB は、RESTful HTTP/JSON API 経由でアクセス可能な、分散型で耐障害性の高い、スキーマ不要のドキュメント指向データベースです。

管理URLは になります。

URLからポート番号の指定を省略する(80番ポートで CouchDBを利用できるようにする)には、/etc/apache2/vhosts.d/default_vhost.include ファイルの末尾に下記の記述を加え、/etc/init.d/apache2 restart でApacheを再起動してください。（その設定をすると、この画面は見られなくなります）

ProxyPass / http://localhost:5984/
